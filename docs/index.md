## Getting Started

```bash
# uuid: Getting Started
// clone
$ git clone https://github.com/mowatermelon/tool.git

// Install dependencies
$ npm i

// Preview docs
$ npm run start

// build libs
$ npm run build

// build docs
$ npm run docs:build
```
