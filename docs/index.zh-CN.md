## 轻松上手

```bash
// 克隆仓库
$ git clone https://github.com/mowatermelon/tool.git

// 按照依赖
$ npm i

// 启动开发
$ npm run start

// 构建功能脚本
$ npm run build

// 构建功能文档
$ npm run docs:build
```
